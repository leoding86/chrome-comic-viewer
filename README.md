# manga-viewer-webextension  
  
A web extension for enhancing user experience for reading manga on website. Support Chrome and Firefox, also support touchscreen  
  
Supported sites:  
readcomics.tv  
readcomiconline.to  
mangapanda.com  
mangastream.com (readms.com)  
eatmanga.com  
kissmanga.com  
mangadoom.cn  
mangapark.me  
(...keep adding)
